QTL Boundary 설명

QTL(Quality Tolerance Limit)은 시험 전체 수준의 품질 허용 한계를 의미
가운데 초록색 원 = 실제 품질 상태 또는 Metric
바깥 원형 고리 = Tolerance Boundary
라벨 = QTL
QTL Boundary를 벗어난 값은 시험 전체 수준에서 품질 문제가 발생 → 추가적인 평가와 조치가 필요
